# JDP System - Produções e Eventos
Sistema completo de gestão e automação para o setor de eventos.

## 🚀 Funcionalidades
- Gestão de equipe e logística de produção.
- Automação de segurança com scripts personalizados.
- Interface otimizada para dispositivos Android.

## 🛠️ Tecnologias
- Android / Webview
- Python (Backend de suporte)
- GitHub Actions (Deploy automático)
